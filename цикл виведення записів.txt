Цикл використовується для виведення записів. 
Використовуючи цикл ми кажем вордпрессу обробити кожну з записів для виведення їх на поточній сторінці і форматувати їх відповідно до того як ми задали. 
Будь який html чи php код, який ми використаємо в циклі буде виведений для кожного з записів.
По замовчуванню в циклі виводяться наступна інформація:
	Заголовок the_title();
	Час the_time();
	Рубрика the_category();
Приклад циклу:
			 

			 <?php if (have_posts()) :  while (have_posts()) : the_post(); ?>
			       <!-- здесь формирование вывода постов, -->
			       <!-- где работают теги шаблона относящиеся к the loop -->
			<?php endwhile; ?>
			<?php endif; ?>
пояснення: Якщо є записи - виводити їх доки вони є. 
Можна ще добавити в кінець: 



            <?php if (have_posts()) :  while (have_posts()) : the_post(); ?>
			       <!-- здесь формирование вывода постов, -->
			       <!-- где работают теги шаблона относящиеся к the loop -->
			<?php endwhile; else: ?>
			<p><?php _e('Вибачте, записів немає'); ?></p>
			<?php endif; ?>
Якщо записи відсутні - виведе Вибачте, записів немає.
