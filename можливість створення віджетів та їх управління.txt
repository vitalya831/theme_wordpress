Віджети - елементи, блоки, які розташовані в певних областях для віджетів, де вони зареєстровані. 
В ці області ми можемо вбудовувати певні віджети (меню, текст, архіви, календар)
Реєстрація області для віджетів:
	в functions.php вставляємо:

					register_sidebar( array(
		'name'          => sprintf(__('Sidebar %d'), $i ),           
		'id'            => "sidebar-$i",
		'description'   => '',
		'class'         => '',
		'before_widget' => '<li id="%1$s" class="widget %2$s">',
		'after_widget'  => "</li>\n",
		'before_title'  => '<h2 class="widgettitle">',
		'after_title'   => "</h2>\n",
	) );


	Він реєструє нову панель для віджетів в адмін-панелі. Параметри на wp-cama.ru
На сайті можливо багато різних областей для віджетів (сайдбар, футер, категорії останні...)
Для кожної з областей необхідно прописувати в functions.php 


					register_sidebar( $args )

Приклад реєстрації:
				 
				 register_sidebar( array(
				'name' => "Права бокова панель сайту",
				'id' => 'right-sidebar',
				'description' => 'Ці віджети відображаються тільки в правій боковій панелі сайту',
				'before_title' => '<h1>',
				'after_title' => '</h1>'
			) );

В адмін-панелі буде відображатись область для віджетів з назвою Права бокова панель сайту
В шаблоні ми можемо вивести цю область за допомогою функції 


				
				<?php
					if ( function_exists('dynamic_sidebar') )
						dynamic_sidebar('right-sidebar');
				?>
